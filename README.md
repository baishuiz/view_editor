### 安装与运行
#### 安装
1. 安装mysql
2. 创建数据库：page
3. 执行server/db文件夹下的page.sql文件
4. 分别在client、server文件夹下运行：
```
npm install
```

#### 运行
1. client目录下执行如下命令
```
npm run dev
```
2. server目录下执行如下命令
```
node app.js
```
3. 启动成功后，访问如下链接：
```
http://localhost:8080/module/home.html
```
